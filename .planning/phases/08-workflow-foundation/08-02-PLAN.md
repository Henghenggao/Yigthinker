From 08-RESEARCH.md (AST validation):
```python
_BLOCKED_CALLS = {"exec", "eval", "__import__", "compile", "getattr", "setattr", "delattr"}
_BLOCKED_MODULES = {"os", "sys", "subprocess", "shutil", "socket", "http"}

def _validate_rendered_script(code: str) -> list[str]:
    issues = []
    try:
        tree = ast.parse(code)
    except SyntaxError as e:
        return [f"Rendered script has syntax error: {e}"]
    for node in ast.walk(tree):
        if isinstance(node, ast.Import):
            for alias in node.names:
                if alias.name.split(".")[0] in _BLOCKED_MODULES:
                    issues.append(f"Blocked import: {alias.name}")
        elif isinstance(node, ast.ImportFrom) and node.module:
            if node.module.split(".")[0] in _BLOCKED_MODULES:
                issues.append(f"Blocked import: {node.module}")
        elif isinstance(node, ast.Call):
            if isinstance(node.func, ast.Name) and node.func.id in _BLOCKED_CALLS:
                issues.append(f"Blocked call: {node.func.id}()")
    return issues
```

```jinja2
#!/usr/bin/env python3
"""{{ workflow_name }} — Auto-generated by Yigthinker v{{ version|default('1') }}"""
from __future__ import annotations
{% block imports %}
import sys
import yaml
import logging
from pathlib import Path
from checkpoint_utils import checkpoint, report_status
{% endblock %}

logger = logging.getLogger("{{ workflow_name }}")

{% block step_functions %}
{% for step in steps %}
def step_{{ step.id }}({{ step.inputs | join(', ') if step.inputs else '' }}{% if step.inputs %}, {% endif %}**config):
    """{{ step.action }}: {{ step.params | tojson }}"""
    {% block step_body scoped %}
    # Action: {{ step.action }}
    {{ step.body | default('pass') }}
    {% endblock %}
{% endfor %}
{% endblock %}

{% block main %}
def main():
    config_path = Path(__file__).parent / "config.yaml"
    with open(config_path, encoding="utf-8") as f:
        config = yaml.safe_load(f)

    logger.info("Starting workflow: {{ workflow_name }}")
    {% for step in steps %}
    {% if step.inputs %}
    result_{{ step.id }} = checkpoint("{{ step.id }}")(step_{{ step.id }})({{ step.inputs | map('regex_replace', '^(.+)$', 'result_\\1') | join(', ') }}, **config)
    {% else %}
    result_{{ step.id }} = checkpoint("{{ step.id }}")(step_{{ step.id }})(**config)
    {% endif %}
    {% endfor %}

    report_status("completed", config)
    logger.info("Workflow {{ workflow_name }} completed successfully")

if __name__ == "__main__":
    main()
{% endblock %}
```

Class `TemplateEngine`:

- `render(self, target: str, context: dict) -> str` -- Load template `f"{target}/main.py.j2"` (for target "python" use "base/main.py.j2"), render with context, call `_validate_rendered_script()` on result. If validation issues found, raise ValueError with the issues list.
- `render_checkpoint_utils(self, context: dict) -> str` -- Render `base/checkpoint_utils.py.j2` with context.
- `render_config(self, context: dict) -> str` -- Render `base/config.yaml.j2` with context. Post-render: scan for credential patterns (`://` with `@`, `sk-` prefix, `password:` with non-vault value). If found, raise ValueError.
- `render_requirements(self, context: dict) -> str` -- Render `base/requirements.txt.j2`. Context must include `dependencies` list computed from step actions.

Module-level constants for AST validation:

- `_ALLOWED_IMPORTS = {"pandas", "numpy", "sqlalchemy", "yaml", "pyyaml", "json", "logging", "pathlib", "datetime", "typing", "requests", "openpyxl", "reportlab", "plotly", "checkpoint_utils", "collections", "re", "math", "decimal", "csv", "io", "dataclasses"}`

Function `_validate_rendered_script(code: str) -> list[str]` -- Parse with `ast.parse()`, walk tree, check imports against _BLOCKED_MODULES, check calls against _BLOCKED_CALLS. Return list of issue strings.
