**Helper functions:**

`_extract_steps_from_history(messages: list[Message]) -> list[WorkflowStep]` -- Per D-12 and D-13:
1. Iterate through messages where `msg.role == "assistant"` and `isinstance(msg.content, list)`
2. For each content block with `block.get("type") == "tool_use"` and `block["name"]` in `_AUTOMATABLE_TOOLS`:
3. Check corresponding tool_result in next user message -- skip if `is_error: True` (filter errors)
4. Build WorkflowStep with `id=f"step_{idx}"`, `action=block["name"]`, `params=block.get("input", {})`, `inputs=[f"step_{idx-1}"]` if not first step
5. Return list

`_normalize_step_params(params: dict) -> dict` -- Coerce common LLM mistakes:
- String `"None"` -> `None`
- String `"True"`/`"False"` -> bool
- Stringified integers -> int
- Stringified floats -> float

`_validate_schedule(schedule: str) -> None` -- Per D-16 (validate at generate time):
```python
try:
    from croniter import croniter
    croniter(schedule)
except (ValueError, KeyError) as e:
    raise ValueError(f"Invalid cron expression '{schedule}': {e}")
```

**Class: WorkflowGenerateTool**
```python
class WorkflowGenerateTool:
    name = "workflow_generate"
    description = (
        "Generate a self-contained Python script from analysis step definitions. "
        "Supports python, power_automate, and uipath targets. "
        "Set from_history=True to auto-extract steps from recent tool calls. "
        "Set update_of to create a new version of an existing workflow."
    )
    input_schema = WorkflowGenerateInput

    def __init__(self, registry: WorkflowRegistry) -> None:
        self._registry = registry
        self._engine = TemplateEngine()
```

`async def execute(self, input: WorkflowGenerateInput, ctx: SessionContext) -> ToolResult`:
1. If `input.from_history and not input.steps`:
   - `steps = _extract_steps_from_history(ctx.messages)`
   - If no steps found, return `ToolResult(tool_use_id="", content="No automatable tool calls found in conversation history", is_error=True)`
2. Else: `steps = input.steps`
3. If `not steps`: return error "No steps provided"
4. Normalize params: `for step in steps: step.params = _normalize_step_params(step.params)`
5. If `input.schedule`: call `_validate_schedule(input.schedule)` -- return ToolResult(is_error=True) on ValueError
6. Compute dependencies: `deps = TemplateEngine.compute_dependencies([s.model_dump() for s in steps])`
7. Build connections list from step params (extract connection-related keys)
8. Build template context dict:
   ```python
   context = {
       "workflow_name": input.name,
       "description": input.description,
       "steps": [s.model_dump() for s in steps],
       "version": version,
       "schedule": input.schedule,
       "connections": connections,
       "dependencies": deps,
       "checkpoint_ids": [s.id for s in steps],
       "max_retries": 3,
       "gateway_url": None,
   }
   ```
9. Render all files:
   ```python
   version_data = {
       "main.py": self._engine.render(input.target, context),
       "checkpoint_utils.py": self._engine.render_checkpoint_utils(context),
       "config.yaml": self._engine.render_config(context),
       "requirements.txt": self._engine.render_requirements(context),
   }
   ```
10. If `input.update_of`:
    - `version_dir = self._registry.update(input.update_of, input.description, version_data, changelog=f"Updated: {input.description}")`
    - Per D-11, previous version untouched
11. Else:
    - `version_dir = self._registry.create(input.name, input.description, version_data)`
12. Write `.gitignore` in workflow dir (parent of version dir): `config.yaml\n*.tmp\n.lock\n`
13. Return `ToolResult(tool_use_id="", content={"output_dir": str(version_dir), "version": version, "files": list(version_data.keys()), "workflow_name": input.name, "target": input.target})`

Wrap entire execute body in try/except Exception returning ToolResult(is_error=True) per codebase error handling pattern.
